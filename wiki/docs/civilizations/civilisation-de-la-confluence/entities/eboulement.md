## Description

L'Éboulement est un obstacle naturel formé par le déplacement de roches ou de terre, bloquant le passage dans une galerie souterraine. Il joue le rôle d'un obstacle physique qui empêche le progrès des explorateurs. Il est mentionné comme étant nombreux, ce qui suggère une série d'obstacles similaires. L'un de ces éboulements révèle une chambre secrète, indiquant qu'il peut aussi être un élément déclencheur de découvertes cachées.

## Chronologie

**Tour 14: Un éboulement bloque le passage dans la galerie, forçant les personnages à faire demi-tour. Cet éboulement cache une chambre secrète que l'un des personnages découvre en glissant sa main à l'intérieur. À l'intérieur de cette chambre, un bol recouvert de poussière est trouvé, laissant passer la lumière du flambeau.**
