## Description

Le couteau est un outil de technologie utilisé par les habitants de la Confluence. Il est principalement utilisé pour évider les poissons, ce qui suggère qu'il est un instrument de subsistance ou de pêche. Sa poignée est mentionnée, ce qui indique qu'il est conçu pour être maniéré facilement. Il est apparu dans un contexte de découverte d'un cadavre, ce qui suggère qu'il peut également être utilisé dans des situations de violence ou de conflit.

## Chronologie

**Tour 12: Un couteau est découvert à côté d'un cadavre, dont la cause de mort semble suspecte. La poignée du couteau est identifiée comme étant de ceux utilisés pour évider les poissons de la Confluence, ce qui évoque un usage quotidien mais aussi une possible implication dans un meurtre.**
