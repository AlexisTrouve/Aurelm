## Description

Les Mineurs sont une caste qui vit principalement dans des grottes souterraines. Ils ont pour habitude d'agrandir ces grottes plutôt que de creuser de nouveaux espaces. Leur présence peut être bénéfique pour les habitants, notamment en offrant une nouvelle source d'eau. Ils sont souvent découverts par hasard, comme dans le cas mentionné, où une nouvelle grotte a été trouvée et utilisée par les habitants.

## Chronologie
